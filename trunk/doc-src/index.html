HyperSQL, aka HSQLDB, is a relational database engine written in Java. Version 1.9.0 is the result of over 3 years develpment since the last major version and has many new features.

This download contains the following files and directories.
bin

This directory contains some Windows utility wrapper scripts, and some HTML pages with HSQLDB embedded as an applet.

build

This directory contains the ant build.xml script plus MSDOS CMD files for build. See the Building Appendix of the User Guide for details.

classes

When the jar is rebuilt, this directory contains the *.class files generated by the ANT build tool. It does not exist in the distribution zip.

doc

A set of HTML , PDF and text documents covering different aspects of HSQLDB and some of its utilities. Within these documents are links to other formats of the same books.

User Guide: doc/guide/index.html

Utilities Guide: doc/util-guide/index.html

chronological list of changes since the release of version 1.9.0 changelist_1_9_0.txt

The license texts for the source and binaries (hypersonic_lic.txt for modified sources originally from the HypersonicSQL project) (hsqldb_lic.txt for sources developed entirely by the HSQLDB Development Group).

The apidocs subdirectory contains the JavaDoc for public classes, including the JDBC documentation.

lib

The jar needed for running HSQLDB (hsqldb.jar) has been prebuild in this directory. The extra zip file is needed only for recompiling hsqldb and is not required for deployment.

src

All source code is in this directory.

testrun

Contains test scripts for the database engine and SqlTool.
