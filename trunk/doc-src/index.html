HyperSQL version 2.5.2 Documentation

HSQLDB (HyperSQL Database) is a relational database engine written in Java. Version 2.5.2 is the result of 10 years development since version 2.0. It offers many features and adheres closely to the latest SQL:2016 and JDBC 4.2 standards.

Additional Resources

Support for HyperSQL is available from http://hsqldb.org/support in various forms, including a mailing list and user forums. The web site features the latest bugfix versions of the software, FAQ and other useful resources.
